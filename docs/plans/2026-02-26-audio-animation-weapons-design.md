# 火柴人游戏增强设计文档

## 概述

为火柴人闯关游戏添加三个核心系统：程序化音效、武侠风格骨骼动画、多武器战斗系统。

## 一、音效系统 (AudioManager)

### 1.1 技术方案

使用 Web Audio API 程序化生成所有音效，无需外部音频文件。

### 1.2 音效类型

| 音效 | 风格 | 实现方式 |
|------|------|----------|
| 背景音乐 (BGM) | 中国风五声音阶 | 振荡器 + 音符序列 + 循环播放 |
| 拳脚攻击 | 短促"呼"声 | 低频噪声 + 快速衰减 |
| 剑攻击 | 金属"刷"声 | 高频噪声 + 带通滤波 |
| 弓箭射击 | 弓弦"崩" + 箭矢"嗖" | 双重音效组合 |
| 死亡 | 下坠撞击 | 频率下滑 + 噪声衰减 |
| 金币收集 | 清脆"叮" | 高频正弦波 |
| 跳跃 | 轻微"腾" | 短促上升音调 |
| 过关 | 欢快上升音阶 | 连续音符递增 |

### 1.3 音量控制

- BGM 音量：30%
- 音效音量：60%

## 二、骨骼动画系统 (SkeletonAnimator)

### 2.1 关节结构

```
        ○ 头
        │
    ┌───┼───┐
   肩    身   肩
    │    体   │
   肘    │    肘     ← 大臂 → 肘 → 小臂
    │    │    │
    └───┼────┘
        │
       髋
      ╱ ╲
    大腿 小腿    ← 髋 → 大腿 → 膝 → 小腿
```

**8个关节点：**
- 左/右肩 (shoulder.left, shoulder.right)
- 左/右肘 (elbow.left, elbow.right)
- 左/右髋 (hip.left, hip.right)
- 左/右膝 (knee.left, knee.right)

### 2.2 动画状态机

| 状态 | 描述 |
|------|------|
| idle | 站立微晃，呼吸感（肩轻微起伏） |
| run | 手臂摆动 + 腿部交替迈步，武侠跑姿（上身略前倾） |
| jump | 起跳：下蹲蓄力 → 腿部蹬直 → 空中收腿；落地：双腿弯曲缓冲 → 恢复 |
| attack_punch | 扭腰转肩 → 手臂快速伸出 → 收回 |
| attack_sword | 反手握剑 → 挥砍弧线 → 剑气拖尾 → 收剑 |
| attack_bow | 双手举弓 → 拉弦蓄力 → 松手放箭 |
| die | 膝盖弯曲 → 身体后仰 → 倒地静止 |

### 2.3 武侠风格动画曲线

```
动画时间轴：
├── 0-30%:  蓄力阶段（重心下沉、关节收缩）
├── 30-60%: 发力阶段（快速展开、身体扭转）
└── 60-100%: 收招阶段（缓慢回归、惯性余韵）
```

## 三、武器系统 (WeaponSystem)

### 3.1 武器配置

| 武器 | 按键 | 射程 | 伤害 | 攻击速度 | 特效 | 弹药 |
|------|------|------|------|----------|------|------|
| 拳脚 | J | 50px | 1 | 快 | 残影拳头 | ∞ |
| 剑 | J | 80px | 2 | 中 | 剑气弧线 + 拖尾 | ∞ |
| 弓箭 | K | 远程 | 1.5 | 慢 | 抛物线飞行 | 10/关 |

### 3.2 按键方案

- `J` — 近战攻击（拳脚或剑，取决于当前选择）
- `K` — 射箭（远程）
- `Q` — 切换近战武器（拳脚 ↔ 剑）

### 3.3 Arrow 类设计

```javascript
class Arrow {
  x, y           // 位置
  vx, vy         // 速度（含重力影响）
  direction      // 飞行方向
  active         // 是否存活

  update() {
    vy += gravity * 0.3  // 抛物线运动
    // 碰撞检测：敌人、平台、边界
  }

  draw() {
    // 箭身 + 箭羽 + 飞行拖尾
  }
}
```

### 3.4 武器视觉特效

**拳脚：** 残影拳头（3帧渐隐效果）

**剑气：** 挥砍弧线 + 渐隐拖尾

**箭矢：** 箭身 + 飞行拖尾

### 3.5 弹药系统

- 每关开始重置为 10 支箭
- 击杀敌人有 20% 概率掉落箭矢（+3 支）
- UI 显示当前箭矢数量

## 五、实现优先级

1. **AudioManager** — 音效系统（独立模块，无依赖）
2. **SkeletonAnimator** — 骨骼动画（需修改 StickMan 类）
3. **WeaponSystem** — 武器系统（需新增 Arrow 类，修改攻击逻辑）
