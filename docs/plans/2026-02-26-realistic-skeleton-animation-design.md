# 写实风格骨骼动画改进设计

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 改进火柴人骨骼动画系统，使所有动作更接近真实人体运动学

**风格:** 写实风格 - 接近真人运动学，注重物理准确性和自然感

**优先级:** 按核心体验优先 - 跑步/跳跃 > 攻击 > 死亡

---

## 一、核心改进原则

**写实人体运动学要点：**
- 关节活动遵循真实解剖学限制
- 运动有惯性和重量感
- 身体各部分协调运动（手臂与腿反向摆动）
- 重心随动作自然转移

---

## 二、跑步动画 (updateRun)

### 当前问题
- 腿部前后对称摆动，缺乏"抬腿-蹬地"节奏
- 膝盖弯曲时机和幅度不正确
- 手臂摆动过于僵硬

### 写实跑步步态周期
```
0% ────────────────────────────────────────────── 100%
├── 0-20%:  着地期 - 脚落地，膝盖微弯吸收冲击
├── 20-50%: 支撑期 - 身体越过支撑脚，另一腿向前摆
├── 50-65%: 蹬地期 - 脚尖蹬地推进，膝盖伸直
└── 65-100%: 摆动期 - 腿向前抬起，膝盖最大弯曲
```

### 关节角度（弧度）
| 关节 | 前摆最大 | 后摆最大 | 说明 |
|------|---------|---------|------|
| 髋关节 | +0.7 (40°) | -0.35 (20°) | 前摆大于后摆 |
| 膝关节 | +1.57 (90°) | 0 | 仅在摆动期弯曲 |
| 肩关节 | ±0.7 | - | 与腿反向 |
| 肘关节 | 0.9-1.2 | - | 保持约90°弯曲 |

### 身体动态
- 上下起伏：3-5px（支撑期最低，摆动期最高）
- 前倾角度：0.1-0.15 弧度
- 周期：20-24帧

## 三、跳跃动画 (updateJump)

### 当前问题
- 起跳蓄力感不足
- 空中姿态过于僵硬
- 落地缓冲不自然

### 三阶段分解

**阶段1: 起跳 (0-30%)**
- 快速下蹲（髋+膝弯曲约 1.7 弧度/100°）
- 手臂向后摆（肩关节 -1.0）
- 爆发蹬地，身体快速伸展
- 身体前倾增加

**阶段2: 腾空 (30-70%)**
- 膝盖收起（髋 0.4，膝 -0.8）
- 手臂向上/前伸（肩关节 -0.8）
- 身体略微蜷缩
- 头部微微上仰

**阶段3: 落地 (70-100%)**
- 腿部前伸准备着地（髋 -0.2）
- 脚尖先着地 → 膝盖弯曲缓冲（膝 1.2）
- 身体前倾稳定重心
- 手臂前摆维持平衡

### 关节关键帧
| 进度 | 髋 | 膝 | 肩 | 肘 | 身体倾斜 |
|-----|-----|-----|-----|-----|---------|
| 0% | 0 | 0 | 0 | 0 | 0 |
| 15% | -0.8 | 1.7 | 0.8 | 0.4 | 0.1 |
| 30% | 0.2 | -0.3 | -1.0 | -0.5 | -0.1 |
| 50% | 0.4 | -0.8 | -0.6 | -0.3 | -0.05 |
| 85% | -0.2 | 1.2 | 0.4 | 0.2 | 0.15 |
| 100% | 0 | 0 | 0 | 0 | 0 |

## 四、攻击动画

### 4.1 拳击 (attack_punch)

**三阶段：预备 → 发力 → 收回**

| 进度 | 动作描述 |
|-----|---------|
| 0-25% | 重心后移，拳头后拉蓄力，身体扭转 |
| 25-60% | 后腿蹬地，腰部回转，手臂直线伸出 |
| 60-100% | 手臂收回防守位置，重心稳定 |

**关键关节：**
- 身体扭转（通过 shoulder 左右差异体现）
- 出拳侧：shoulder = 1.2 (前伸), elbow = 0 (伸直)
- 后拉侧：shoulder = -0.8 (后收), elbow = 0.8 (弯曲)

### 4.2 剑攻击 (attack_sword)

**三阶段：预备 → 斩击 → 收剑**

| 进度 | 动作描述 |
|-----|---------|
| 0-25% | 剑举过头顶，身体侧转蓄力 |
| 25-55% | 身体回转带动剑下劈，前脚迈出 |
| 55-100% | 剑惯性运动后停止，恢复警戒姿态 |

### 4.3 射箭 (attack_bow)

**四阶段：搭箭 → 拉弓 → 放箭 → 回位**

| 进度 | 动作描述 |
|-----|---------|
| 0-30% | 侧身站立，一手持弓前伸 |
| 30-60% | 后手拉弦至脸侧，身体微后仰 |
| 60-80% | 松开弦，后手顺势后摆 |
| 80-100% | 双手放下，恢复站姿 |

## 五、死亡动画 (updateDie)

### 当前问题
- 倒地过程太快
- 缺乏失去意识的重量感
- 身体各部位反应不协调

### 三阶段分解

**阶段1: 中击 (0-20%)**
- 被击中部位瞬间收缩
- 身体向反方向后仰
- 头部后仰
- 手臂无意识张开
- 关节：hip = -0.3, knee = 0.5, bodyTilt = -0.2

**阶段2: 失去平衡 (20-50%)**
- 膝盖发软，身体开始下坠
- 手臂乱抓（失去控制）
- 重心不稳，开始倾斜
- 头部无力下垂
- 关节：hip = 0.3, knee = 1.2, bodyTilt = 0.3

**阶段3: 倒地 (50-100%)**
- 身体重重摔向地面
- 先膝盖着地，再上半身倒下
- 手臂自然垂落地面
- 头部最后着地
- 关节：hip = 0.2, knee = 1.0, bodyTilt = 1.3

### 关键细节
- 倒地速度由慢到快（重力加速效果）
- 落地时有轻微弹跳
- 动画周期延长至 80-100帧
